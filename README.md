# 🎬 NovaStream

**NovaStream** is a modern movie streaming web application where users can watch movies and series, discover new content, rate films, and interact with the platform through personalized recommendations and social features.

---

# 🚀 Features

## 🔑 Core Features

These features provide the fundamental functionality of the platform.

* **User Authentication**

  * Secure user registration and login
  * Role-based access control

* **Movie Catalog**

  * Browse a collection of movies and series
  * View movie details

* **Video Streaming**

  * Watch movies directly within the platform

* **Watch History**

  * Automatically track movies watched by users

* **Search**

  * Quickly find movies by title

---

## ⭐ Advanced Features

###  Personalized Movie Recommendations

NovaStream suggests movies to users based on:

* previously watched movies
* user ratings
* preferred genres

---

###  Leaderboards

Includes dynamic leaderboards showing:

* users who watched the most movies
* top reviewers

---

###  Movie Rating System

Users can rate movies using a star-based system.

---

###  Trending Movies Algorithm

NovaStream dynamically calculates and displays trending movies based on:

* number of views
* ratings
* watchlist additions
* recently added

This allows users to easily discover **popular content on the platform**.

---

###  Continue Watching

Users can resume videos exactly where they stopped watching.

---

###  Watchlist System

Users can save movies to a personal watchlist to view later.

---

###  Smart Search and Filtering

Advanced filtering allows users to discover movies based on:

* genre
* rating
* release year
* popularity

---

## 👥 User Roles

## Normal User

Regular users interact with the platform through streaming and community features.

Permissions include:

* Watch movies
* Rate movies
* Write reviews
* Add movies to watchlist
* View personalized recommendations

---

## Admin

Administrators manage the platform and its content.

Permissions include:

* Add or remove movies
* Edit movie information
* Manage users
* Delete inappropriate reviews

---

# 🛠️ Requirements

### Frontend

* A responsive UI for desktop and mobile devices
* Pages for authentication (login / register)
* Movie browsing interface with catalog and details pages
* Video player interface for streaming movies
* User profile area including watch history and watchlist
* Leaderboards and recommendation sections
* Search bar with filtering options

Technologies:

* React
* HTML5
* CSS3
* JavaScript

---

### Backend

* User authentication and session management
* Handling movie data and metadata
* Managing user interactions (ratings, reviews, watchlists)
* Processing recommendation logic
* Tracking watch history and playback progress
* Admin functionality for managing movies and users
* Serving video content for streaming

Technology:

* Python (3.10 or higher)

---

### Database

The system will store information such as:

* User accounts
* Movies and their metadata
* Ratings and reviews
* Watchlists
* Watch history
* Leaderboard data

Technology:

* PostgreSQL

---

### Media Storage

Options include:

* Local server storage
* Cloud storage services
